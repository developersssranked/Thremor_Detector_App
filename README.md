# Тремор-анализатор

Моё первое мобильное приложение под Android. Меряет дрожание руки по показаниям акселерометра и показывает результат на экране — RMS, пик, примерную частоту и общий уровень тремора (Норма / Лёгкий / Умеренный / Сильный).

Сделано на **Kotlin** + **Jetpack Compose**.

---

## Что умеет

- Слушать акселерометр (`TYPE_LINEAR_ACCELERATION`, если есть, иначе обычный с фильтром гравитации).
- Считать в реальном времени:
  - модуль вектора ускорения `a = √(ax² + ay² + az²)`;
  - **RMS** по окну из 128 последних замеров;
  - **пиковое** значение;
  - примерную **частоту** колебаний в Гц (через zero-crossings).
- Показывать уровень тремора цветом (зелёный → жёлтый → оранжевый → красный).
- Рисовать график волны (Canvas в Compose).
- Кнопки **Старт / Стоп / Сброс**.
- Не теряет данные при повороте экрана (ViewModel).

---

## Скриншот

Экран сверху вниз:
1. Заголовок и подсказка.
2. Большая цветная карточка с названием уровня.
3. Три карточки: RMS, Пик, Частота.
4. Полоска «Интенсивность».
5. График волны.
6. Кнопки управления.

## Как запустить

### Что нужно поставить

- **Android Studio** (Hedgehog или новее — нужна поддержка compileSdk 36).
- **JDK 11** (обычно ставится вместе со студией).
- Телефон с Android 7.0+

### Шаги

1. Скачать проект:

   ```bash
   git clone https://github.com/<ваш-логин>/first_app.git
   cd first_app
   ```

2. Открыть папку `first_app` в Android Studio (`File → Open...`).


4. Подключить телефон по USB (включить «Отладку по USB» в настройках разработчика)

5. Нажать зелёную кнопку **Run** (▶) или `Shift + F10`.

## Как пользоваться

1. Открыть приложение.
2. Положить телефон на ладонь или вытянутую руку.
3. Нажать **Старт**.
4. Держать руку 10–20 секунд.
5. Посмотреть на цифры и цвет карточки.
6. Нажать **Стоп** (или **Сброс**, если хочется обнулить всё).

### Что значат цифры

- **RMS** — общая «сила» дрожания, м/с². Чем больше — тем сильнее тремор.
- **Пик** — самое сильное колебание за окно, м/с².
- **Частота** — сколько раз в секунду рука «дёргается», Гц.

### Уровни

| Цвет | Уровень | RMS (м/с²) |
|---|---|---|
| 🟢 Зелёный | Норма | < 0.15 |
| 🟡 Жёлтый | Лёгкий | 0.15 – 0.6 |
| 🟠 Оранжевый | Умеренный | 0.6 – 1.5 |
| 🔴 Красный | Сильный | > 1.5 |

> Пороги подобраны экспериментально, на разных телефонах могут отличаться.

## Тестирование на эмуляторе

Сам по себе эмулятор не дрожит. Чтобы данные пошли:

1. Запустить эмулятор.
2. Открыть `...` (Extended controls) рядом с окном эмулятора.
3. Перейти во вкладку **Virtual sensors**.
4. Двигать ползунки `Accelerometer x / y / z` — приложение будет на это реагировать.

---

## Известные ограничения

- Частота считается простым методом (zero-crossings), а не FFT — это не очень точно.
- Пороги уровней одинаковые для всех телефонов, без калибровки.
- Только один экран, истории замеров нет.
- UI только на русском.
